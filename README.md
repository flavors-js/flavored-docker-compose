# flavored-docker-compose

[docker-compose](https://docs.docker.com/compose/) plugin for [Flavors CLI](https://github.com/flavors-js/flavors-cli).<br>

## Install

```text
$ npm install -g flavors-cli flavored-docker-compose
```
